Panduan Upload Gambar ke GitHub

Cara paling stabil: Upload gambar manual ke folder images/galeri/ di repo GitHub Anda, lalu gunakan path relatif di admin panel ini.

Langkah-langkah:

  1. Buat folder images/galeri/ di repo GitHub Anda
  2. Upload gambar (JPG/PNG) ke folder tersebut via GitHub web atau git push
  3. Di form tambah galeri di bawah, pilih tab "Path GitHub"
  4. Masukkan path: images/galeri/nama-file.jpg

Contoh path yang benar:

images/galeri/sampling-urea.jpg
images/galeri/sealing-pestisida.jpg

Tips: Optimalkan gambar sebelum upload (max 500KB) menggunakan TinyPNG agar website tetap cepat.
